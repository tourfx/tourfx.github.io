Техническая поддержка

Уважаемые пользователи!
Техническая поддержка программы TourFX Агент осуществляется по электронной почте agent@tourfx.ru только для пользователей, имеющих лицензии Lite и Pro на программу.
Желательно подробно описать проблему и последовательность действий, которые к ней приводят. Также обратите внимание, в программе имеется встроенный механизм снятия скриншотов экранных форм. Просто нажмите клавишу PrintScreen (или PrtSc) на клавиатуре. Сверху справа появится меню, с помощью которого вы можете сохранить скриншот в файл или сразу отправить его на электронный адрес службы техподдержки.

Перед обращением в службу техподдержки просьба ознакомиться со следующим списком проблем и их решений:

  • Ошибка соединения с базой данных
  • Не работает печать документов в заявке
  • Слишком мелкий шрифт в программе
  • Программа не запускается после обновления
  • Как перенести базу данных с одного компьютера на другой

Ошибка соединения с базой данных

Попробуйте выполнить следующие действия:

  • Проверьте, что компьютер имеет доступ по локальной сети к компьютеру-серверу, на котором установлен Microsoft SQL Server.
  • Отключите брандмауэр Windows на компьютере, где установлен Microsoft SQL Server. Брандмауэр может блокировать входящие соединения. Обратите внимание, что даже если вы вручную выключили брандмауэр, некоторые программы могут его самостоятельно без вашего участия включить, и программа TourFX Агент опять не сможет подключиться к базе данных. Если отключение решает проблему, можно добавить правило, разрешающее входящие соединения на порт 1433 (порт Microsoft SQL Server), и включить обратно брандмауэр.
  • Временно отключите антивируc.
  • Если ошибка возникает сразу после установки Microsoft SQL Server, удалите его, а затем установите и настройте его заново, очень внимательно следуя инструкции. Неправильное или неточное выполнение любого из шагов установки или настройки SQL-сервера приводят к проблеме с подключением программы к базе данных.

Не работает печать документов в заявке

Вся печать в программе TourFX Агент осуществляется через обычные документы Word (формат DOCX) и Excel (формат XLS). Убедитесь, что у вас установлен Microsoft Office версии 2007 и выше или LibreOffice версии 4.1 и выше. Microsoft Office 2003 не подходит для печати договора без установленного пакета поддержки формата DOCX. Microsoft Office в Стартовой редакции (иногда предустановлен на новых компьютерах) любой версии не подходит для использования с программой TourFX Агент. Рекомендуется в таких случаях использовать бесплатный офисный пакет LibreOffice.

Слишком мелкий шрифт в программе

Если ваш монитор имеет высокое разрешение скачайте и установите в ту же папку программу из специального дистрибутива. Также размер шрифта можно изменить в Пользовательских настройках.

Программа не запускается после обновления

На некоторых компьютерах антивирусы и другие защитные программы иногда блокируют процесс установки обновления, и возможна ситуация, когда установка файлов новой версии была произведена не до конца. В таком случае нужно:

  • перезагрузить компьютер (программу не запускать после перезагрузки),
  • скачать полный дистрибутив программы с сайта tourfx.ru и установить его в ту же папку, в которую программа была установлена, поверх старых файлов, ничего не удаляя.

Затем рекомендуется отключить в настройках автоматическую систему обновлений и устанавливать новые версии из полного дистрибутива.

Как перенести базу данных с одного компьютера на другой

Для однопользовательской версии можно перенести целиком папку TourFX. Все данные хранятся внутри нее.

Для многопользовательской версии используйте инструкцию по переносу базы Microsoft SQL Server.
