Программа позволяет выставлять счета клиентам (как организациям, так и физическим лицам), контролировать сроки оплат, регистрировать платежи.

Создание счета

Для оформления и печати счета введите сумму оплаты на форме "Оплаты клиентов" и нажмите кнопку "Счет". Откроется форма
счета с реквизитами для оплаты.

Регистрация оплаты

При поступлении оплаты на расчетный счет можно отметить этот факт в соответствующем Счете. Будет автоматически создан
документ "Поступление на р/с", а сам счет переведен в состояние Оплачен.

Автоматическая регистрация оплаты

При импорте банковской выписки программа автоматически ищет номер счета в Назначении платежа операции и отмечает счет как оплаченный.

Настройка нумерации

Правило нумерации счетов может быть настроено с помощью формы "Нумерация документов". Для использования в качестве
компонента номера доступны только "Дата документа" и "Счетчик". По умолчанию, номер формируется из двух последних цифр
текущего года и четырех цифр счетчика.

QR-код в Счете

В шаблоне счета для вывода QR-кода используется тег [QRCode] .

В QR-код попадают следующие реквизиты:

  • Название компании
  • ИНН
  • КПП
  • Номер расчетного счета
  • Название банка
  • БИК
  • Корреспондентский счет
  • Сумма к оплате
  • Назначение платежа

Размер QR-кода можно настраивать с помощью числового параметра в теге.
Если указано значение от 1 до 2 (например, 1.5), то размер будет масштабироваться относительно минимального. Таким образом [QRCode(1)] выведет изображение минимально возможного размера, а [QRCode(1.25)] – в масштабе 125%. Дробная часть указывается через точку, а не через запятую. [QRCode(1.7)] – правильно, [QRCode(1,7)] – неправильно.
При использовании значений (целых чисел) от 100 и более выводимое изображение будет иметь указанный размер в миллиметрах. Например, [QRCode(200)] выведет изображение QR-кода размером 200х200 мм.
По умолчанию (без указания числового параметра) используется масштаб 150%, т.е. [QRCode] эквивалентен [QRCode(1.5)].

Экспорт счета в 1С

При нажатии кнопки "Экспортировать" создается файл с данными счета в формате 1С EnterpriseData версии 1.4. Далее этот файл можно импортировать в 1С с помощью обработки.

Форма настройки экспорта вызывается из контекстного меню кнопки Экспортировать нажатием правой клавиши мыши над этой кнопкой.
