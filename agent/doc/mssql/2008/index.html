Установка Microsoft SQL Server 2008 Express Edition

  1. Скачайте Microsoft SQL Server 2008 Express Edition. На странице загрузке после нажатия на красную кнопку Скачать появится список файлов. Нужно выбрать файл с именем SQLEXPRWT_x64_RUS.exe, если у вас 64-битная версия Windows, или файл SQLEXPRWT_x86_RUS.exe, если 32-битная.

  2. Запустите скаченный файл. Для установки Microsoft SQL Server вы должны обладать правами администратора операционной системы.
    На первом экране выберите первый пункт – "Новая установка или добавление компонентов к существующей установке"

  3. Появится экран с условиями лицензии:

    Для продолжения установки необходимо принять условия лицензионного соглашения и нажать кнопку "Далее".

  4. На экране выбора компонентов снимите флаг с пункта "Репликация SQL Server" и нажмите "Далее".

  5. На следующем экране выберите первый пункт "Экземпляр по умолчанию":

  6. На экране "Конфигурация сервера" выберите значение "Авто" в колонке "Тип запуска" для службы "Браузер SQL Server":

  7. Выберите пункт "Смешанный режим" и введите пароль sa (системного администратора):

    Запомните введенный пароль. Он понадобится вам при создании базы данных и для обслуживания базы данных.

  8. На экране выбора параметров отчетов об ошибках нажмите кнопку "Далее":

  9. Начнется процесс установки:

  10. На следующем экране появится сообщение о завершении установки:

    Нажмите кнопку "Закрыть".

Настройка Microsoft SQL Server 2008

После завершения установки "Microsoft SQL Server 2008 Express Edition" необходимо выполнить следующие действия для
настройки установленного SQL-сервера:

  1. Запустите "Диспетчер конфигурации SQL Server" ("Пуск"  →  "Все программы"  →  "Microsoft SQL Server 2008 R2"  →  "Средства настройки"  →  "Диспетчер конфигурации SQL Server"):

  2. В левой части экрана разверните пункт "Сетевая конфигурация SQL Server" и выберите его подпункт "Протоколы для
    MSSQLSERVER". В правой части экрана выберите пункт "TCP/IP", и, нажав правой кнопкой, выберите в появившемся меню пункт "Включить".

  3. В левой части экрана выберите пункт "Службы SQL Server". В правой части экрана выберите пункт SQL Server (MSSQLSERVER), и, нажав правой кнопкой, выберите в появившемся меню пункт "Перезапустить".

Настройка Microsoft SQL Server 2008 завершена.
