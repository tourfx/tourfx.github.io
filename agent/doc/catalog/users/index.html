Карточка пользователя

Для однопользовательской версии карточка пользователя содержит следующую информацию:

  • фамилия, имя пользователя,
  • фамилия, имя, отчество пользователя в родительном падеже,
  • фамилия, имя пользователя на английском языке,
  • офис, в котором работает данный сотрудник,
  • e-mail и телефон,
  • основное юрлицо, на которое по-умолчанию оформляются заявки,

  • номер и дата доверенности (для договоров).

Для многопользовательской версии карточка пользователя содержит следующую информацию:

  • фамилия, имя пользователя,
  • пароль для входа в систему,
  • основная роль в программе,
  • дополнительные роли,
  • офис, в котором работает сотрудник,
  • e-mail и телефон,
  • фамилия, имя, отчество пользователя в родительном падеже,
  • фамилия, имя пользователя на английском языке,
  • наименование должности в именительном и родительном падежах (для договоров),
  • основное юрлицо, на которое по-умолчанию оформляются заявки данным пользователем,
  • номер и дата доверенности (для договоров).

С помощью ролей определяется, какие действия может совершать пользователь в системе. Например, безналичные оплаты может проводить только пользователь с ролью "Бухгалтер", заявки может создавать "Менеджер", просматривать все отчеты может "Директор", добавлять новых пользователей может только "Администратор".
