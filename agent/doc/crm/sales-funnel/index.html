Воронка продаж

Воронка продаж предназначена для анализа эффективности продаж каждого менеджера в графическом виде. Она
показывает прохождение клиентами этапов воронки за определенный период времени. В результат попадают только те
клиенты, по которым работа завершена продажей тура или отказом клиента. Для каждого этапа можно посмотреть причины, по
которым клиенты отказались от заказа тура.
