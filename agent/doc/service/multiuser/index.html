Переход на многопользовательскую версию

Для переноса данных необходимо выполнить следующие действия:

  1. Установите и настройте SQL сервер по инструкции.

  2. Запустите программу в однопользовательской версии.

  3. Откройте форму "Базы данных" в меню Настройки.

  4. Выберите переключатель Многопользовательская.

  5. В поле Сервер введите название компьютера, на котором установлен SQL Server.

  6. Нажмите кнопку "Создать" и в появившемся окне введите пароль SA. Программа создаст необходимые базы данных на SQL-сервере.

  7. Нажмите кнопку "Перенос данных". Начнется процесс перемещения данных из однопользовательской БД в многопользовательскую.

  8. Нажмите кнопку "Подключиться".
