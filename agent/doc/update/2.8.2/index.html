Измения в версии 2.8.2

Новый журнал заявок

В журнале заявок добавлены несколько дополнительных колонок: Услуги, Город вылета, Счета, Счета от т/о и другие.

Состав отображаемых колонок можно настроить с помощью формы "Настройка таблицы". Для ее вызова нужно нажать правой кнопкой мыши над таблицей и в контекстном меню выбрать "Настройка таблицы...".

Переключатель блокировки ширины колонки позволяет зафиксировать ширину колонки. Измененные вручную ширины колонок сохраняются при последующем повторном открытии журнала.

Выбранные значения фильтров на верхней панели также сохраняется при последующем открытии журнала заявок (между рабочими сессиями).

Дополнительные фильтры по типу услуги, по стране и по туроператору доступны по кнопке .

Когда хотя бы один из этих фильтров активен, иконка меняет цвет: . Значение этих фильтров не сохраняется между рабочими сессиями.

Возврат по безналу

Документ предназначен для автоматического проведения возврата клиенту при импорте банковской выписки. В поле "Назначение платежа" необходимо указать такое же значение, как и в отправленном в банк платежном поручении.
