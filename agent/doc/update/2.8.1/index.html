Измения в версии 2.8.1

Импорт данных компании из текстового файла

Можно скопировать данные компании в текстовом виде в буфер обмена. Тогда при добавлении новой компании в справочник большинство полей в карточке будет заполнено автоматически.

Баланс счета SMS-шлюза

При отправке SMS-сообщений по списку рассылки программа теперь показывает текущий баланс счета SMS-шлюза.

Цветом показывается (примерно), достаточно ли средств для выполнения рассылки. Зеленый баланс - достаточно средств для отправки всех сообщений. Красный - скорее всего, не все сообщения будут приняты для отправки.
