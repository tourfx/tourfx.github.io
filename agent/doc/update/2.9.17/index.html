Изменения в версии 2.9.17

Банковские реквизиты в карточке туроператора

В карточку туроператора добавлены поля для заполнения банковских реквизитов:

  • Номер расчетного счета
  • Название банка
  • БИК
  • Корреспондентский счет

В шаблонах заявки (договоры и другие документы, формирующиеся из заявки) можно использовать следующие теги для вывода этих данных:

  • Туроператор.БанкРек.РасчетныйСчет
  • Туроператор.БанкРек.Банк
  • Туроператор.БанкРек.БИК
  • Туроператор.БанкРек.КоррСчет

Почтовый адрес в карточке туроператора

Добавлено поле "Почтовый адрес" и соответствующий тег Туроператор.ПочтовыйАдрес для вывода в шаблонах.
