Измения в версии 2.8

SMS
  • Отправка SMS из программы: напоминания, поздравления, уведомления
  • Управление списками рассылок
  • Отправка информационных SMS-сообщений по списку рассылки
Заявки
  • Оформление заявок от юрлиц
  • Формирование счетов клиентам
  • Регистрация счетов от туроператоров
  • Импорт банковской выписки, автоматическое разнесение оплат
  • Справочник клиентов-юрлиц
  • Редизайн услуги "Авиаперелет"
  • Город вылета
CRM
  • Расширенные отчеты CRM: сравнение воронок продаж, воронки продаж по текущему состоянию
  • Поиск клиента в базе при добавлении нового
  • Редактирование чек-листа по заявке
  • Отчет по заканчивающимся загранпаспортам

Изменения в версии 2.8.1

  • Импорт данных компании из текстового файла
  • Баланс счета SMS-шлюза

Изменения в версии 2.8.2

  • Новый журнал заявок
  • Возврат по безналу
  • Улучшена работа программы в сетях с нестабильным качеством соединения
