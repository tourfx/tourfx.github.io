Обновление программы в macOS

1) Скачайте дистрибутив программы последней версии по этой ссылке.

2) Убедитесь что программа TourFX закрыта и запустите скаченный файл двойным кликом мыши. Откроется окно, в котором нужно перетащить иконку программы TourFX в папку Applications.

3) При появлении следующего окна подверждения необходимо нажать кнопку "Заменить".
